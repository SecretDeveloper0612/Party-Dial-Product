# Party-Dial Client Mobile App

This is the mobile application for Party-Dial clients, built with React Native and Expo.

## Features

- **Premium Home Screen**: Explore event categories and featured venues.
- **Search & Filter**: Find venues by location, type, and capacity.
- **Venue Details**: View photos, amenities, and pricing.
- **Appwrite Integration**: Ready for backend communication.
- **Expo Router**: Modern file-based navigation.

## Getting Started

1.  Navigate to the directory:
    ```bash
    cd ClientMobile
    ```
2.  Install dependencies (if not already):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npx expo start
    ```

## Development

- **Colors**: Shared constants in `constants/Colors.ts`.
- **Pages**: Located in `app/`.
- **Components**: Reusable UI elements in `components/`.
- **Backend**: Client setup in `lib/appwrite.ts`.
